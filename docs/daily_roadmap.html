Design Scoring Framework
3-4 hours
MEDIUM
Create the mathematical foundation for PropertyIQ scoring.
  • Define scoring weights: Infrastructure (40%), Safety (35%), Economic (25%)
  • Create 0-100 point scale with clear ranges
  • Design sub-component scoring for each category
  • Build scoring functions with proper normalization
  • Add confidence intervals based on data freshness
class PropertyScorer: def __init__(self): self.weights = { 'infrastructure': 0.40, 'safety': 0.35, 'economic': 0.25 } def calculate_score(self, data): infrastructure_score = self.score_infrastructure(data) safety_score = self.score_safety(data) economic_score = self.score_economic(data) total = (infrastructure_score * self.weights['infrastructure'] + safety_score * self.weights['safety'] + economic_score * self.weights['economic']) return min(100, max(0, total))
